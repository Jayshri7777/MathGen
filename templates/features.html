About MathGen

Welcome to Math Worksheet Generator! This tool was built to help students, teachers, and parents create custom math worksheets in seconds.

Our mission is to make learning and practicing math more accessible and customizable. By leveraging the power of generative AI, we can create problems tailored to specific grades, topics, and difficulty levels, all with the click of a button.

This project uses a Flask backend, a SQLite database for user management, and the Google Gemini API for content generation. We hope you find it useful!
